### 关于硬件设计

硬件PCB设计参考了[ROS机器人仓库](https://github.com/fan-ziqi/My_ROS_Robot)的电路和布局。

主控采用STM32F103C6T6,一个TB6612实现驱动两路电机，板载MPU6050以及CH340N串口芯片实现与电脑通信。
